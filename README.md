# 三角洲行动每日密码查询插件

一个简单的 AstrBot 插件，用于查询游戏《三角洲行动》的每日密码。

## 功能

- 支持多个触发词，避免误触
- 支持地图：零号大坝、长弓溪谷、航天基地、巴克什、潮汐监狱

## 触发词

| 触发词 | 说明 |
|:-------|:-----|
| `/粥密码` | 主命令 |
| `/每日密码` | 别名 |
| `/三角洲密码` | 别名 |

## 安装

1. 在 AstrBot WebUI 插件商店搜索「三角洲行动密码」安装
2. 或手动将插件文件夹放入 `data/plugins/` 目录

## 使用

在聊天中发送任意一个触发词即可：
```
/粥密码
/每日密码
/三角洲密码
```

## 数据来源

[Tmini.net](https://tmini.net) - 三角洲行动每日密码 API
